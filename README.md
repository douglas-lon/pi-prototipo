# Pi-Prototipo

## Requerimentos

* Python 3.8+
* Flask modulo
* Flask_wtf
* WTForms

## Instalação Flask
No terminal digite:
```
pip install flask
```

## Instalação Extenções Flask
No terminal digite:
```
pip install flask-wtf
pip install email_validator
```

## Iniciar a aplicação
Abra o terminal na pasta da aplicação e digite:
```
python run.py
```
Você conseguerá ver ela no link:
http://localhost:5000/
